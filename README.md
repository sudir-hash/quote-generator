# quote-generator

This is a simple random quote Generator.
The site is deployed with Netlify 
[Click here to view](https://gilded-starburst-f5f80a.netlify.app/)

## How to run?

# Clone the repo
```
  git clone https://github.com/sudir-hash/quote-generator
```
# To run dev server
```
cd quote-generator/
npm i
npm run dev
```
